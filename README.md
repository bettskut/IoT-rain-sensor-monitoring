# IoT Rain & Environmental Monitoring System

Developed an IoT rain monitoring system using ESP32, Raindrops Detection KIT, and DHT22, with real-time data displayed on a modern web dashboard.

## 🚀 Features
* **Real-Time Visualization**: High-precision data monitoring via a "System Command Center" dashboard.
* **Database Integration**: Automated data logging to **MongoDB** for persistent storage and historical analysis.
* **Responsive Design**: Modern UI built with React and Tailwind CSS.

## 🛠️ Tech Stack
* **Microcontroller**: ESP32
* **Sensors**: Raindrops Detection KIT, DHT22
* **Backend**: Node.js, Express
* **Database**: MongoDB (Local Instance)
* **Frontend**: React.js, Tailwind CSS
